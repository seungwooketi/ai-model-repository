Model Download API

API Documentation

Each micro-service can use this API to download the model easily from the Model Repository. This server provides multiple AI models which can classify, predict the future status. This repository server provides simple API to access and download the right model for a local edge.

This REST API have four types of parameters: model_name, model_version, model_split, and split_number. 

										
											

												
Example
	
												    
 This example demonstrates downloading a specific AI model from DECENTER AI model repository.
												    

													
    
														
  • http://host_address/model_download?model_name=UC4_FaceDetection&model_version=0.1&model_split=Split_No&split_number=0
    • 
													

											

										
									
										
											

												
1. Base URL
	
													
 Base URL is http://hostaddrress/model_download

													
Ex) http://182.252.132.39:5000/model_download

											

										
									
										
											

												
2. Model Name
	
													
 First of all, you need to know the right model name among registered models. 

													
Ex) model_name=UC4_FaceDetection

											

										
									
										
											

												
3. Model Version
	
													
 Select a proper version of a specific AI model.

													
Ex) model_version=0.1

											

										
									
										
											

												
4. Model split information
	
													
There are two options:[Split_No, Split_Yes] for model_split. "Split_No" is a default option.

													
Ex) model_split=Split_No

											

										
									
										
											

												
5. Split Number
	
													
 The split number is required to download one of the unit of the split model set. For non-split models, split_number should always be fixed at zero. 

													
Ex) split_number=1
